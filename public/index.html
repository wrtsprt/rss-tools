Feed Enrichment

Some RSS feeds only contain headlines or abstracts of the original article. It's only a limited amount of fun reading those in a news reader app (e.g. Reeder). See below for hardcoded and dynamic solutions.

Dynamic

try (ALPHA) feed enhancers. create your own configuration how to enrich an RSS feed.

Static
